Dataset Overview

The dataset consists of sevearal files with lists of "urls" and post ids and user ids. You can view these by downlorading the dataset and understand why it was done this way by viewing or downloading dhe explainer notebook. All of the information can be boiled down to 2 files.

It took several hours to collect the data, primarily to avoid the webscraper to be detected as a bot. And when we were detected we had to "lay low" for a few hours before scraping again. So the data was collected over the span of three days.

posts_data_unique.csv

  • Description: This is the file containing the entirety of posts and comments from DarkNetMarkets
  • Size: 64.3 MB
  • Variables: type, id, parent_id, title, author, content
  • Date: 19.4.2025 - 21.4.2025

links_data.csv

  • Description: This file is the scraped content of the direct link of all the unique links found in posts_data_unique's content
  • Size: 135.4 KB
  • Variables: address, alias, content
  • Date: 2.5.2025
