Introduction

The purpose of this project is to analyze and understand the dynamics of the DarkNetMarkets subdread on Dread, one of the largest forums on the dark web. Through network and text analysis, we aim to map the relationships, discussions, and key actors in this hidden corner of the internet.

Understanding the Dark Web

The dark web operates differently from the surface web we use daily. Websites use onion routing (Tor protocol) to hide their location and identity, making traditional search engines ineffective. Unlike Google's domain-based indexing, dark web content can only be discovered through three main methods: brute forcing addresses, user submissions, or scraping known sites for new onion keys. This opaque environment creates a need for centralized forums like Dread (likely named as a play on "Dark Reddit") where users can share information and links.

What We Did

We scraped and analyzed the DarkNetMarkets subdread on Dread, one of the most active communities discussing illicit marketplaces. Our project includes:

  • Network analysis of user interactions and relationships
  • Text analysis of discussions to identify popular marketplaces and key topics
  • Identifying dominant users, markets, and vendors in discussions

Why We Did It

The dark web remains fragmented and poorly understood by researchers. By analyzing Dread - currently the largest dark web forum - we can gain unique insights into this hidden ecosystem. Our analysis of the DarkNetMarkets subdread provides:

  • A snapshot of discussions around illicit marketplaces
  • Understanding of how communities react to law enforcement actions
  • Potential to identify key actors, scam patterns, and emerging threats

Dataset Overview

This project explores data collected from DarkNetMarkets, an online forum where users discuss various marketplaces on the dark web. Our goal was to analyze the types of posts, the people involved, and the external links being shared within this unique digital environment.

What’s in the Dataset?

The dataset was built over three days (April 19–21, 2025) using a custom-built web scraper. Because we wanted to avoid detection and bans by the forum’s anti-bot measures, we had to slow down scraping when flagged — adding to the challenge. Dread is 7 years old and DarkNetMarkets have existed since day one. This means that we have data dating all the way back to February 15. 2018. In the end, the data was condensed into on large file with 197.516 posts and comments from the subforum.
